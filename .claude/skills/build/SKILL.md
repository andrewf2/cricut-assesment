# /build

Build the shared library and then the weather app. Report any errors with suggested fixes.

## Instructions

1. Run `cd weather-app && npx ng build lib`
2. If the lib build **fails**, report the errors and suggest fixes. Stop here.
3. If the lib build **succeeds**, run `npx ng build weather-app`
4. If the app build **fails**, report the errors and suggest fixes.
5. If both builds **succeed**, report success with the output summary.
